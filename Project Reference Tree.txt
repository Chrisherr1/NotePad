Stack

Express + EJS + MySQL
Passport.js (Google OAuth)
CSRF protection + Session management(Csrf-sync + Cookie-session)
HTTPS with local SSL(Mkcert)

Key Dependencies
express, ejs, mysql2, passport-google-oauth20
express-session, express-mysql-session
csrf-sync, cookie-session

Routes

Route	     Purpose
/auth/*	     Google OAuth flow
/dashboard	 Main app interface
/notes	     CRUD operations
/csrf-token	 Token endpoint

Setup
bash
npm install
# Add .env: DATABASE, GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, SESSION_SECRET
mysql -u root -p < schema.sql
npm start
